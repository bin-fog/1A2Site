# НЕЙМАРК.АБИТУРИЕНТЫ
Сервис, решающий проблемы абитуриентов, позволяя им сформировать список образовательных программ и рекомендуя лекции с мероприятиями на основе их достижений, предпочтений:  персональную анкету, список достижений, формирование рекомендаций, автоматический парсинг достижений из дипломов и результатов ЕГЭ, формирование актуальных образовательных программ.
